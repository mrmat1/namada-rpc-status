# Namada rpc status

## Features

- Monitor rpcs status
- Monitor indexers status
- Checking your rpc

Live at : https://rpc-monitor.duongnm.online

## Installation

```bash

git clone https://github.com/mrmat1/namada-rpc-status.git
npm install -g yarn

#Run server :
cd namada-rpc-status/server
yarn install
node server.js
# http://localhost:4003

# run client
cd namada-rpc-status/client
# run in produce mode
yarn build-prod
yarn preview

# http://localhost:4001
```
